\# Brekotkin Jumpscare Mod



Inspired by \*"1/10000 Chance for Withered Foxy Jumpscare Every Second"\*, this mod adds a random full-screen jumpscare of Dmitry Brekotkin with sound effect.



\## Features

\- Random appearance configurable via chance (default: 1 in 10,000 ticks)

\- Adjustable display duration and sound volume

\- Client-side only — works in any world, solo or multiplayer



\## Configuration

Go to: `Mod Config → BrekotkinConfig`



\## Installation

1\. Install from Steam.

2\. Subscribe to the mod on \[Steam Workshop]( no link yet :( ).

3\. Enable it in-game under "Mods".
